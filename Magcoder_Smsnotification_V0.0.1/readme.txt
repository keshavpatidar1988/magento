Please read this file before installing extension.
This module is for vacinfomedia sms api only for Indian mobile notifications!!

1. Unzip package.
2. Copy folders , files and paste it into the root of your shopping cart.
3. Merge files and folders (If you are using this extension first time, else make sure you have not modified the files, if you are updating code).
4. Login to admin refresh the cache, after refreshing cache you will be see "404  Management" tab under CMS.
5. Go to SMS manager tab under that go to SMS settings tab and do configuration of the API details provided by VACINFOMEDIA . 
6. If you see 404 error on enabling extension logout and login into shopping cart.
7. Once the API has been configire perform test notification from SMS settings and if all good start creating messages from manage sms tab under sms manager.

Features.

1. Admin can configure/update/test the VACINFOMEDIA api and make sure they are working fine or not.
2. Perform testing of API on any mobile number and message from admin.
3. Feature to add mobile number to get notifed when new order has been created.
4. Can setup notifications for New order, Invoice, Shipment, Complete order. 
5. Activate/ Deactivate notification sent or not from admin. 
6. Compatible to 6788 patch.
7. Compatible to default theme form version 1.6.x.x to 1.9.2.4 
8. Smsnotifictions.log all error messages regarding related to orders has been log in smsnotifications.log under var/log folder.
